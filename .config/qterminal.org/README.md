# QTerminal configurations

Code:
https://github.com/lxqt/qterminal

These are the main customizations I've made:

**Open a maximized window**
```ini
[General]
#...
LastWindowMaximized=true
#...

[MainWindow]
#...
pos=@Point(0 0)
size=@Size(1920 1020)
#...
```

**A better font size**
```ini
fontSize=18
```
